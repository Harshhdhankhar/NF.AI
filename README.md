# NeuroFlow AI - AI-Powered Mental Wellness Platform

A modern, responsive website for NeuroFlow AI, featuring cutting-edge design with 3D animations, dark/light mode toggle, and interactive elements.

## 🌟 Features

- **Modern Design**: Futuristic UI with gradient backgrounds and 3D animations
- **Dark/Light Mode**: Smooth theme switching with localStorage persistence
- **3D Animations**: CSS 3D transforms, rotating cubes, floating cards
- **Interactive Elements**: Tilt effects, hover animations, particle system
- **Responsive Design**: Mobile-first approach with breakpoints for all devices
- **Neuron Network**: Animated neural network visualization
- **Accessibility**: WCAG compliant with keyboard navigation support
- **Performance**: Optimized with passive event listeners and debounced functions

## 🚀 Quick Start

1. Clone or download the project files
2. Open `index.html` in a modern web browser
3. For development, use a local server:
   ```bash
   python3 -m http.server 8000
   ```
4. Visit `http://localhost:8000` to view the website

## 📱 Sections

1. **Hero Section**: Logo, tagline, and animated background
2. **Features**: 24/7 AI Chatbot, Crisis Control, Therapy Plans, etc.
3. **How It Works**: Animated workflow visualization
4. **Benefits**: Flip cards with hover effects
5. **USP**: Comparison slider and 3D highlights
6. **Team**: Animated profile cards
7. **Contact**: Interactive form and contact information

## 🛠️ Technologies

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Grid, Flexbox, 3D transforms, animations
- **JavaScript ES6**: Modern vanilla JS, no frameworks
- **Google Fonts**: Inter and Poppins typography

## 🎨 Design Features

- **3D Animations**: `transform: perspective()`, `rotateX/Y/Z()`
- **Particle System**: Dynamic floating particles
- **Smooth Transitions**: CSS transitions and animations
- **Interactive Hover**: Tilt effects and glow animations
- **Mobile Responsive**: Breakpoints at 768px and 480px

## 🔧 Customization

### Animations
- Modify animation durations in CSS
- Adjust particle count in JavaScript
- Customize 3D rotation effects

## 📊 Performance

- Optimized animations with `transform` and `opacity`
- Debounced scroll and resize handlers
- Passive event listeners
- Reduced motion support for accessibility
